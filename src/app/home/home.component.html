A fun game for family and friends

How to play Create a new game, or join one that someone created. Recommended for 4-6 players Each player will start with a phrase that will get 'telephoned' around. Eg: 'Golden retriever' The next player will translate the phrase to emoji The next player will translate back to english Keep going round robin until all the messages have been translated!
